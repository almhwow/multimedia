Images

These three images of the moon are all in different formats.

First image is a .jpg, a format that contains compressed image data.

Second image is a .png, a raster-graphics file format that supports lossless data compression.

Third image is an HDR, a picture taken with a set of techniques used to reproduce a greater range of luminosity than that which is possible with standard photographic techniques.
